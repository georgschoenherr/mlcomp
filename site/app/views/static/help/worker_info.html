Run Environment

All programs are run in the cloud on a standard Linux distribution, specifically, a 32-bit Amazon EC2 machine running Fedora Release 8 Linux (kernel 2.6.21). The table below shows the following packages installed by default. Let us know if you would like us to install additional packages.

ProgramVersion
C/C++ gcc/g++ 4.1.2
Java Sun JDK 1.6.0_16
Scala 2.7.6
Perl 5.8.8
Python 2.5.1
Ruby 1.8.6
R 2.8.0
Octave 3.0.3
clisp 2.43-3
guile 1.8.5-1
Ocaml 3.10.0-8

numpy and scipy are pre-installed for Python. If your program requires libraries that we do not have installed, consider using the CDE tool. See the demo program for more information.

Your program may use up to 1.5GB of memory, run for at most 24 hours, use up to 10GB of disk space. It may not use the network.
