Run Environment

All programs are run in the cloud on a standard Linux distribution, specifically, a 32-bit Amazon EC2 machine running Linux (Ubuntu 11.04, kernel 2.6.38). The table below shows the following packages installed by default. Let us know if you would like us to install additional packages.

Program Version Additional Packages
C/C++ (gcc/g++) 4.5.2
Java (Sun) 1.7.0
Scala 2.9.0
C# (mono) 2.6.7
Perl 5.10.1
Python 2.7.1 numpy, scipy
Ruby 1.8.7
R 2.12.1
Octave 3.2.4
Lisp (clisp) 2.48
Scheme (guile) 1.8.7
OCaml 3.11.2
Haskell (ghc) 6.12.3

If your program requires libraries that we do not have installed, consider using the CDE tool. See the demo program for more information.

Your program may use up to 1.5GB of memory, run for at most 24 hours, use up to 10GB of disk space. It may not use the network.
